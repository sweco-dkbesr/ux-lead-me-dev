LEAN UX = SCRUM + UX

Lean UX kan betragtes som udvidelse af SCRUM med metoder og artefakter fra UX verdenen.

I Lean UX skelnes der mellem FUNDAMENTALT UX og RETNINGSBESTEMT US.

Fundamentalt UX udføres typisk sjælden, det består til gengæld af større opgaver, såsom Personas, Design Sprints etc.

Se forneden, hvordan Lean UX adskiller sig fra Vanilla SCRUM og gå videre til beskrivelser af de enkelte metoder og produkter.
